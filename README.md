# Insecurity Talk

Project create for tech talk on BallastLane about security on rails projects

* Ruby version: 3.1.0

+ PostgreSQL: 16

+ Setup:
  + change database credentials on `config/database.yml`
  + rails db:create
  + rails db:migrate
  + rails db:seed
  + rails serve
